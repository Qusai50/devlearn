Linux Tutorial

Linux is an open-source operating system kernel that powers servers, cloud infrastructure, Android, and developer workstations worldwide. Learning the Linux command line (the terminal) is essential for any developer or system administrator.

Linux Intro

Unlike Windows or macOS graphical interfaces, Linux provides a powerful text-based interface called the terminal or shell. The default shell is usually Bash. Commands are typed and executed immediately.

command [options] [arguments]

File System

Linux uses a tree-like file system that starts at the root directory /. Important directories:

PathPurpose
/Root directory
/homeUser home directories
/etcConfiguration files
/varLogs and variable data
/tmpTemporary files

ls - List Files

Lists files and directories in the current location.

ls
ls -l       # Long format
ls -a       # Show hidden files
ls -la      # Both combined

cd - Change Directory

Moves between folders.

cd foldername
cd ..       # Up one level
cd ~        # Home directory

pwd - Working Directory

Prints the full path of the current directory.

pwd

mkdir - Make Directory

Creates a new folder.

mkdir new_folder
mkdir -p path/to/folder  # Create nested

rm - Delete Files

Removes files or directories.

rm file.txt
rm -r folder/      # Recursive delete
rm -rf folder/     # Force recursive (careful!)

cp - Copy

Copies files or folders.

cp file1.txt file2.txt
cp -r folder1/ folder2/

mv - Move / Rename

Moves or renames files/folders.

mv old.txt new.txt
mv file.txt /home/user/docs/

cat - View File

Displays the contents of a file.

cat file.txt
cat file1.txt file2.txt > combined.txt

grep - Search

Searches for text patterns inside files.

grep "error" log.txt
grep -i "warning" *.log
grep -r "TODO" .

chmod - Permissions

Changes file access permissions.

chmod 755 script.sh
chmod +x script.sh
NumberPermission
7read + write + execute
6read + write
5read + execute
4read only

Pipes & Redirection

Combine commands and control input/output.

ls -l | grep ".txt"      # Pipe output to grep
cat file.txt > out.txt   # Redirect output
command 2> errors.log    # Redirect errors

Bash Scripting

A Bash script is a text file with multiple commands that can be executed together.

#!/bin/bash
echo "Hello World"
NAME="DevLearn"
echo "Welcome to $NAME"

Save as script.sh, then:

chmod +x script.sh
./script.sh
💡 Tip: Use the man command (e.g., man ls) to see detailed documentation for any command.
