Python Introduction

Python is used for web development, software development, mathematics, system scripting, data science, machine learning, and more. It works on different platforms and has a simple syntax similar to the English language.

Python Syntax

Python indentation indicates a block of code. Python uses new lines to complete a command.

if 5 > 2:
    print("Five is greater than two!")

Python Variables

Variables are containers for storing data values. Python has no command for declaring a variable.

x = 5
y = "Hello, World!"

# Casting
x = str(3)    # '3'
y = int(3)    # 3
z = float(3)  # 3.0

Python If...Else

Python supports conditional statements to execute code based on different decisions.

a = 33
b = 200
if b > a:
    print("b is greater than a")
elif a == b:
    print("a and b are equal")
else:
    print("a is greater than b")

Python Loops

Python has two primitive loop commands: while and for.

# While loop
i = 1
while i < 6:
    print(i)
    i += 1

# For loop
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

Python Functions

A function is a block of code which only runs when it is called.

def greet(name):
    return "Hello, " + name

print(greet("Alice"))

Python Lists

Lists are used to store multiple items in a single variable.

mylist = ["apple", "banana", "cherry"]
mylist.append("orange")
mylist.remove("banana")
print(mylist[0])

Python Dictionaries

Dictionaries are used to store data values in key:value pairs.

car = {
    "brand": "Ford",
    "model": "Mustang",
    "year": 1964
}
print(car["brand"])
car["year"] = 2020

Python File Handling

File handling is an important part of any web application.

# Read file
f = open("demofile.txt", "r")
print(f.read())
f.close()

# Write file
f = open("demofile.txt", "w")
f.write("Now the file has more content!")
f.close()
