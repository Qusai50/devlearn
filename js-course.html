JavaScript Tutorial

JavaScript is the world's most popular programming language. It is the programming language of the Web.

Try it Yourself

JS Introduction

JavaScript can change HTML content, attribute values, styles, and hide/show elements. It is lightweight, interpreted, and just-in-time compiled.

JS Syntax

JavaScript syntax is the set of rules for constructing programs:

// Variables
var x = 5;
let y = 6;
const z = x + y;

JS Variables

Variables are containers for storing data values. Use let for variables that change, and const for constants.

let firstName = "John";
const pi = 3.14;

JS Operators

JavaScript uses arithmetic, assignment, comparison, logical, and type operators.

OperatorDescription
+Addition
-Subtraction
*Multiplication
/Division
%Modulus
===Strict equal

JS Functions

A JavaScript function is a block of code designed to perform a particular task.

function greet(name) {
  return "Hello, " + name;
}
console.log(greet("Alice"));

JS Objects

Objects are variables too, but they can contain many values.

const person = {
  firstName: "John",
  lastName: "Doe",
  age: 50,
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};

JS Events

HTML events are "things" that happen to HTML elements. JavaScript lets you execute code on events.

JS Strings

Strings are for storing and manipulating text. You can use single or double quotes.

let text = "Hello World";
let length = text.length;        // 11
let part = text.slice(0, 5);     // Hello

JS Arrays

Arrays are used to store multiple values in a single variable.

const cars = ["Saab", "Volvo", "BMW"];
cars.push("Audi");               // Add item
cars.forEach(car => console.log(car));

JS Loops

Loops can execute a block of code a number of times.

for (let i = 0; i < 5; i++) {
  console.log(i);
}

const fruits = ["apple", "banana"];
for (let fruit of fruits) {
  console.log(fruit);
}

JS DOM

The DOM (Document Object Model) is a programming interface for web documents.

document.getElementById("demo").innerHTML = "Hello";
document.querySelectorAll("p").forEach(p => p.style.color = "red");

JS JSON

JSON (JavaScript Object Notation) is a lightweight data interchange format.

const obj = JSON.parse('{"name":"John", "age":30}');
const text = JSON.stringify(obj);

JS Async

Async programming allows operations to run without blocking the main thread.

async function fetchData() {
  let response = await fetch("https://api.example.com/data");
  let data = await response.json();
  console.log(data);
}

JS Closures

A closure gives you access to an outer function's scope from an inner function.

function makeCounter() {
  let count = 0;
  return function() {
    return ++count;
  };
}
const counter = makeCounter();
console.log(counter()); // 1

ES6 Features

ES6 introduced let/const, arrow functions, classes, template literals, destructuring, and modules.

const square = x => x * x;
const [a, b] = [1, 2];
class Animal {
  constructor(name) { this.name = name; }
}
const msg = `Hello ${name}`;
