SQL Tutorial

SQL is a standard language for storing, manipulating, and retrieving data in databases.

Try it Yourself

SQL Introduction

SQL stands for Structured Query Language. It lets you access and manipulate databases. SQL became a standard of the American National Standards Institute (ANSI) in 1986.

SQL Syntax

Most SQL statements end with a semicolon.
SELECT column1, column2 FROM table_name;

SQL SELECT

The SELECT statement is used to select data from a database.

SELECT * FROM Customers;
SELECT CustomerName, City FROM Customers;
SELECT DISTINCT Country FROM Customers;
